# Конструктор бургеров

Как запустить:
```bash
$ git clone git@github.com:artemchiruhin/react-burger.git react-burger
$ cd react-burger
$ npm i
$ npm run start
```

Как запустить тесты:
```bash
$ npm run test
$ npm run cypress:open
```

Как задеплоить:
```bash
$ npm run deploy
```

Проект доступен по [https://artemchiruhin.github.io/react-burger/index.html](ссылке)
